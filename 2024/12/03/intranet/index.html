环境准备

配置服务端

  1. 安装并部署 ZeroTier 服务端
  1. 访问 http://ip:3443 进入管理页面, 默认账号：admin 密码：password
  2. 使用 登录后，点击选项卡中的 Add Network 来创建一个网络。
  3. 点击上方 IPv4 Assign Mode ，再勾选 Auto-assign from IP Assignment Pool
  4. 复制下括号里的网络ID

配置客户端

  1. ZeroTier 下载
  2. 将 planet 文件复制到:
    linux: /var/lib/zerotier-one/
    windows: C:\ProgramData\ZeroTier\One
  3. 重新启动 ZeroTier service zerotier-one restart
  4. 加入服务端的网络 zerotier-cli join 服务端的网络ID
  5. 前往服务端的控制页面，勾选 Authorized
